Forms

Form elements include text boxes, drop-downs, check boxes, radio buttons, and other elements that are used to collect data from the user.

Text Boxes

Form Layouts

There are two types of form layouts: Basic and Inline. On small viewports, both types render the same — vertically, with labels above the fields.

Basic Forms

For larger viewports, Basic forms do not change, unless you make use of the inline-label class on individual fields within the Basic form (the drop-down fields in the following demo serve as an example).

Basic Form

				<input type="text" id="basicTextBox" placeholder="Placeholder" aria-describedby="basicHelpText">
				<p id="basicHelpText" class="help-text">This is additional text to help describe the expected input.</p>

Inline Forms

Inline forms allow labels to be aligned horizontally with their respective fields or for multiple fields to be aligned horizontally.

Use the inline-form class on a wrapper element and the form-field class on its direct children.

Inline Form

{{instructionText}} 
<form action="#" method="post">
	<fieldset>
		<legend>Inline Form (Inline Labels)</legend>

		<div class="inline-form">
			<div class="form-field">
				<label for="inlineTextBox">Text Box Label</label>

				<div>
					<input type="text" id="inlineTextBox" placeholder="Placeholder" aria-describedby="inlineHelpText">
					<p id="inlineHelpText" class="help-text">This is additional text to help describe the expected input.</p>
				</div>
			</div>

			<div class="form-field">
				<label for="inlineTextarea">Textarea Label</label>
				<textarea name="" id="inlineTextarea" cols="50" rows="5" placeholder="Placeholder text&hellip;"></textarea>
			</div>

			<div class="form-field">
				<label for="inlineSelect">Drop-down Label</label>

				<select id="inlineSelect">
					<option value="">Select an option</option>
					<option value="">Option 1</option>
					<option value="">Option 2</option>
					<option value="">Option 3</option>
				</select>
			</div>
		</div>
	</fieldset>
</form>

Inline Form (Multiple Inline Fields)

When multiple fields are used, labels are not displayed inline anymore. However, like Basic forms, the inline-label class can be used to force a label inline, as seen in the e-mail address field below.

Inline Form

{{instructionText}} 
<form action="#" method="post">
	<fieldset>
		<legend>Inline Form (Multiple Inline Fields and Inline Labels)</legend>

		<div class="inline-form">
			<div class="form-field multiple-fields">
				<div class="inline-label">
					<label for="email">E-mail Address</label>
					<input type="email" id="email">
				</div>

				<div class="inline-label">
					<input type="checkbox" id="signup">
					<label for="signup">Sign me up for the newsletter</label>
				</div>
			</div>
		</div>
	</fieldset>
</form>
Inline Form (Multiple Inline Fields and Inline Labels)
